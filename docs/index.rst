FixedPoint Documentation
========================

This package provides a class to enable fixed point number calculations.

Example:

    >>> from fixedpoint import FixedPoint
    >>> a = FixedPoint(3.45, 'Q4.4')
    >>> a
    FixedPoint(3.4375, 'Q4.4')
